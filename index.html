Hey There!


beginner.js showcases the basics of javascript. The goal is to keep it simple and allow you to start discovering your coding potential.


Write your code in the top box, aka the text editor. Running the code will output the results below in the console.


Note: The javascript method console.log() is needed in order to display any output. Push the "run code" button to see it working!

Lesson One: Strings


Strings are used for storing and changing text. They consist of zero or more characters and are written inside of quotation marks.


Note: You can use single ' ' or double " " quotes.


Print the string "Hello World!" (or any string you'd prefer) in the console.


Lesson Two: Numbers


Numbers can be manipulated using operators.


Some you may already know are:

  • addition +
  • subtraction -
  • multiplication *
  • division /

Note: The order of operations is implimented when multiple operators are used.


Print the number 15 in the console. Use different combinations of numbers and operators.


Lesson Three: Variables


Variables are containers used to store values. They are mainly used to simplify code by acting as placeholders.


A variable is declared with the let keyword and assigned with the = operator.


Note: You can name a varible anything you choose (as long as it isn't already a javascript keyword).


Run the example code given to get an idea of how variables work. Then, change up the variable names and values and print the results to the console.


Lesson Four: Arrays


Arrays are used to hold multiple values.


Notice how the example array contains a string, number, and a variable. Arrays can store all the data types used in these lessons.


Square brackets [ ] are used to create a new array.


Create your own string, number, and varible to store in an array.


Lesson Five: Functions


Functions are dynamic approaches used to simplify code. They can be created once and be used repeatedly with different input.


Input values are called arguments and are inserted in parenthesis () when calling the function. The keyword return must be used to output a value.


The example is a simple function that can add any two numbers together. Try changing the numbers to get different results.
